Definitions / Formulas





Speed vs Velocity


Speed:

It might sound quite intuitive to you, but speed measures "how fast an object is moving". Nothing crazy to see here...
But something you have to remember is that speed doesn't care about the direction of movement. It is a scalar (and not vectorial) quantity.

(recall scalars vs vectors)


Velocity

On the other hand, velocity is a vectorial quantity, which means that it does care about the direction of the movemement (since vectors have a direction)



As a result, two cars moving opposite directions at same speed don't have the same velocity !


Be careful about units!

In Physics, some units are used and some are not. That's the rules.

(read more about International System of Units)



Therefore, for speed/velocity, always remember to convert the following units:

  • Distance is measured in meters (no kilometers, no miles, no centimeters, etc..)
  • Time is measured in seconds (no minutes, no hours, etc...)
  • Therefore, speed/velocity is measured in meters per second (no miles per hour, not kilometers per hour etc...)

If some data is given to you in other units, you should always remember to first convert them into meters/seconds before touching anything!
